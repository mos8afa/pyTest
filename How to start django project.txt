Create virtual environment and project :
    1- Create virtual environment --> virtualenv name
    2- Activate the virtual environment --> name\scripts\activate
    3- Install django in the virtualenv --> pip install django
    4- Create empty project --> django-admin startproject name .
    5- Run the server --> python manage.py runserver
_______________________________________________________________________
How to create admin:
    python manage.py createsuperuser 
_______________________________________________________________________
How to create app:
    python manage.py startapp name
